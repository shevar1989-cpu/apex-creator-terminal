# Apex Creator Terminal — Demo Script

## 1. Introduction
“Welcome — this is the Apex Creator Terminal, the command‑line gateway to the Apex Genesis Engine.  
It generates structured, metadata‑driven project templates for simulations, tools, and modules.”

---

## 2. Launch the CLI
Run:

```
apex
```

Say:
“This is the global CLI entry point. It works anywhere on the system.”

---

## 3. Generate a Simulation Template
Run:

```
apex new template --tag simulation
```

Say:
“ACT filters templates by tag. Here I’m generating a simulation module.”

Walk through the wizard:
- Project name  
- Options  
- Preview  
- Confirmation  

---

## 4. Show the Preview
Explain:
“Every template includes metadata — name, description, tags, options, and output structure.  
ACT shows a preview before generating anything.”

---

## 5. Generate the Project
Confirm generation.

Show output:
- File creation logs  
- Summary  
- Success message  

---

## 6. Open the Project
Open the generated folder in VS Code.

Say:
“ACT produces clean, structured, ready‑to‑extend project scaffolds.  
This is the foundation for the Apex Genesis Engine ecosystem.”

---

## 7. Close
“This is v0.1.0 — the first public release of the Apex Creator Terminal.  
More modules, templates, and GUI tools are coming next.”
